# AeroAustral Líneas Aéreas

AeroAustral Líneas Aéreas es un servicio de areolínea que agenda y proveé transporte aéreo en aviones comerciales a todo público. Para poder facilitar el agendamiento de transporte aéreo a sus usuarios la empresa cuenta con un portal de clientes en línea, desde el cuál pueden realizar todo el proceso tramitario y transaccional para agendar un asiento en un avión en la fecha y hora más conveniente disponible.

## Portal de clientes

El portal de clientes se encuentra en línea, encargado de proveerles la capacidad de agendar un vuelo. Para ello el protal les solicita la información de el o los pasaportes de el o los usuarios y, tras comprovar sus papeles, solicitarles el pago por el servicio de vuelo. Debido a esto el propio portal almacena dentro de su base de datos las propias reservas, conteniendo los datos del avión, asiento, hora de salida y destino de cada reserva, los datos de los pasaportes de los usuarios y los recibos de los pagos para cada agendamiento, cada uno incluyendo los datos bancarios de cada usuario que utilizó para la transacción.
