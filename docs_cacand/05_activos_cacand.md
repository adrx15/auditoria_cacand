# Activos de Información y Riesgos de la Industria

Para una aerolínea como AeroAustral, la continuidad del servicio y la confidencialidad de los datos fronterizos y financieros son el núcleo del negocio.

## Identificación de Activos Críticos
1. **Activo A - Base de Datos de Pasajeros (Pasaportes e Identidad):** Crítico para regulaciones de aviación civil e inmigración.
2. **Activo B - Pasarela de Pagos e Historial de Transacciones:** Datos críticos bajo estricto cumplimiento normativo PCI-DSS.
3. **Activo C - Sistema de Despacho y Asignación de Vuelos:** Esencial para la disponibilidad operativa básica (evitar que los aviones queden en tierra).
